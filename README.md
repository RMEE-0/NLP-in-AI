Purpose: NLP web application using Python for text analysis.

Libraries: NLTK (tokenization), spaCy (tokenization, POS, NER), TextBlob (sentiment), Gradio (web interface).

Function (nlp_analyze): Tokenizes text, tags parts of speech, extracts named entities, and analyzes sentiment.

Output: Displays results in readable markdown with tokens, POS tags, entities, and sentiment scores.

Interface: Gradio app with a textbox input and markdown output for instant text analysis.
